- [Windows](https://github.com/rauenzi/BetterDiscordApp/releases/download/0.3.0/BandagedBD_Windows.exe)
- [MacOS](https://github.com/rauenzi/BetterDiscordApp/releases/download/0.3.0/BandagedBD_Mac.zip)
